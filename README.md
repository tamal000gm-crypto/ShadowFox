# ShadowFox Internship Projects – AIML & Data Analytics

This repository contains the projects completed during my internship at ShadowFox, focusing on Machine Learning and Data Analytics.

Throughout this internship, I worked on real-world datasets and implemented end-to-end data analysis and machine learning workflows using Python and Google Colab.

## Projects Included

### 1️⃣ Boston House Price Prediction
- Built a Linear Regression model to predict housing prices.
- Performed data preprocessing and handled missing values.
- Split dataset into training and testing sets.
- Evaluated model performance using Mean Squared Error (MSE) and R² Score.
- Visualized actual vs predicted values.

### 2️⃣ Superstore Sales & Profit Analysis
- Conducted Exploratory Data Analysis (EDA) on business sales data.
- Analyzed total sales and profit performance.
- Explored category-wise and state-wise profitability.
- Identified monthly sales trends.
- Generated business insights through data visualization.
